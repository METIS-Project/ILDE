Observation of collaborative work
Problem:

In collaborative face-to-face settings, how can the teacher gather information about group work?

Solution:

In order to gather information about the collaborative process when it happens in a face-to-face setting, it is possible to observe directly the collaboration. This is accomplished by the assessor, taking notes about all the relevant aspects of collaboration: interactions between group members, work strategy, how conflicts are solved, etc.

On the other hand, there is software available to support this type of assessment. For instance, the ILOCA software tool can be used to facilitate the task of recording information about interactions between students.

Observation can be used to intervene in the activity and correct situations that hinder the achievement of the learning objectives. On the other hand, the observations can be used by the teacher to produce feedback reports, for the students to be aware of the way they are working.

Application guidelines:

In order to include this assessment technique, is is recommended:

  • This pattern defines the assessment technique used. Therefore, it cannot be combined with any other pattern that also defines an assessment technique. However, patterns that propose some other additional configuration can be combined with this one.
  • In the assess task, there should be instructions for the assessor about how to observe collaboration. For instance, you can include Rubrics, to show what the assessment criteria are.
  • It is necessary to make sure that the assess task is carried out at the same time as the assessed collaborative learning activity.
